리액트 js - 자바스크립트 기반
fucntion, object ,variable
리액트는 자바스크립트, 새로운 것 배울 필요가 없다. 
리액트를 하면서 동시에 자바스크립트 기술을 향상 시키는 것
구조가 간단하다
요소별, 컴포넌트별로 나눠서 작업할 수 있다. 
nav, header, numbers, grid
그룹별로 쪼개서 작업할 수 있다. 
모든 걸 작은 요소로 쪼개서 작업, 마무리하면 가져다가 다른 곳에 쓸 수 있다. 
jsx 는 리액트로 html을 쓰는 방법
리액트는 단방향 데이터플로우를 가지고 있다. 
데이터는 항상 일정한 장소에 위치해있고 그 장소에서만 변경할 수 있다. 
데이터는 위치한 장소에 있고 상태가 변했을 경우, 데이터는 그대로 있고 UI가 업데이트 되는 것임
즉, 데이터가 UI를 변경하는 것
데이터 -> ui : 단방향
데이터가 변하면 ui 가 업데이트 된다.
데이터가 안변하면, ui도 안변한다.
ui에서 데이터를 바꿀 수 는 없다. 
데이터 -> 데이터 변경 -> ui 변경
커뮤니티가 정말 바앧하고 크다. 거대한 라이브러리, 오픈소스, 질문과 답 ... 
리액트는 프레임워크가 아니라 UI 라이브러리
앵귤러랑은 쓸 수 없고 파이썬, 루비랑 쓸 수 있고 view 이기 때문에 섞어 쓸 수 있다. 
model, view, controller에서 리액트는 view이기 때문에 나머지는 원하는대로 골라서 쓸 수 있다.
